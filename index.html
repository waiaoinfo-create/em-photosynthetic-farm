壹、EM菌與光合菌概論

EM菌（Effective Microorganisms）

  • 起源：日本琉球大學比嘉照夫教授
  • 組成：光合菌群 + 乳酸菌群 + 酵母菌群 + 放線菌群 + 絲狀菌群，5大類10屬56餘種微生物
  • 特性：兼性厭氧/好氧，群體協同
  • 功能定位：綜合型土壤改良、飼料添加、環境除臭、水質淨化

光合菌（Photosynthetic Bacteria, PSB）

  • 核心菌種：紫色非硫菌——沼澤紅假單胞菌（Rhodopseudomonas palustris）
  • 特性：厭氧光照或好氧黑暗皆可生長；以光為能源
  • 功能定位：固氮溶磷、分解H₂S與NH₃-N、分泌植物生長因子、促進根系
  • 台灣主要推廣：財團法人食品工業發展研究所（食工所, FIRDI）

貳、核心差異釐清

EM菌是複合菌群（包含光合菌），功能廣泛但各菌比例不明；光合菌是單一菌群，功能專一、可純化培養、自製成本更低。建議策略：「光合菌為主（根系強化）、EM菌為輔（堆肥+除臭）」。

比較項目EM菌光合菌
取得難度容易（市售多）需向食工所或特定來源
自製難度低（糖+水+菌種）中（需光照管理）
成本效益極高極高
土壤改良全面性佳專一性強（固氮溶磷）
水產應用良好極佳（降氨氮）
養殖除臭極佳良好
葉面噴施佳極佳（促根系）
堆肥加速極佳普通
保存性較差（液體衰退快）較佳（厭氧性穩定）
安全性經廣泛驗證農糧署公告安全菌種

3.2 擴培步驟

第一步：準備

  • 容器：20L透明PE桶（或寶特瓶、玻璃瓶），清洗後以75%酒精消毒
  • 水：RO水或煮沸後放涼自來水（除氯），水溫25~35°C
  • 環境：乾淨通風處，避免雜菌污染

第二步：混合

  1. 桶中加入5L RO水
  2. 加入培養基粉末200g，搖晃溶解
  3. 加入母菌200ml
  4. 加RO水至桶口近滿（盡量減少空氣殘留）
  5. 旋緊瓶蓋

第三步：培養

  • 陽光充足處（直曬可，避免溫度 > 40°C）
  • 每日搖晃1~2次，使沉澱菌體均勻受光
  • 溫度：25~35°C 最佳
  • 顏色變化：淡粉→粉紅→深紅→紫紅
  • 週期：夏季7~10天，冬季14~21天
  • 完成判別：暗紅/紫紅色，醬油/烏梅香氣（非臭味）

第四步：保存

  • 移至陰涼處（低溫冷藏更佳）
  • 密封避光，可存放3~6個月
  • 使用前搖勻

肆、EM菌自製SOP

材料（40公斤產量）數量約略成本
紅糖2~4kg80~160元
水（除氯自來水/井水）40L0元
EM活菌自產劑500g150~250元
50L以上塑膠桶1個200~300元（首次）
每公斤成本（不含桶）約5.8~10.3元

操作步驟：

  1. 桶中加入35°C溫水40L
  2. 加入紅糖2~4kg攪拌溶解
  3. 加入EM自產劑500g攪拌均勻
  4. 密封，置於25~35°C環境
  5. 每日開蓋放氣+攪拌1~2次
  6. 約7天完成（低溫延長），產氣連續減少即完成
  7. 活菌數：每毫升 ≥ 100億CFU（1個月內）

4.2 紅糖擴繁法（簡易版）

市售EM原液1L + 紅糖1kg + 水20L → 密封發酵5~7天

⚠ 注意：菌數逐代衰減，建議最多擴繁3代即需重購母菌。乳酸菌在常溫液體中難以長時間存活，第二個月活菌下降約30%，第三個月下降約50%。

伍、農場四大應用領域與具體用量

5.1 種植業應用

A. 土壤改良（基肥）

項目EM菌光合菌
整地前稀釋500倍，200~300L/分地稀釋300倍，150~200L/分地
休耕期每周一次×4周每周一次×3周
連作障礙田稀釋200倍，300L/分地/月稀釋100倍全面浸泡

預期效果：有機質提升0.3~0.8%（6~12月）、有益菌群增加10~100倍、pH趨向中性

B. 育苗與定植

  • 浸種：稀釋100~200倍，浸泡15~30分鐘（硬殼種子2小時）
  • 育苗介質：500倍稀釋澆灌，每周一次
  • 定植澆根：EM菌200~300倍，每株200~500ml；光合菌300~500倍，每株300~500ml
  • 每分地用量：約20~40L菌液

C. 生長期葉面噴施

作物稀釋倍數頻率注意
葉菜類300~500倍7~10天一次採收前3天停用
果樹類200~300倍10~14天一次避開盛花期
花卉300~500倍7~10天一次促進開花數
根莖類500~800倍僅幼苗期膨大期禁用

⚠ 開花期關鍵規範

  • 光合菌會刺激營養生長（長葉不長花），盛花期必須暫停
  • EM菌可在開花期以低濃度（500~800倍）使用
  • 建議用菌時機：開花前2周停止，謝花後恢復

5.3 畜禽養殖

項目EM菌用量
飲水添加500~1000倍稀釋，連續飲用
飼料發酵每噸飼料：EM菌2~5L + 糖2kg + 水100~150L，發酵3~7天
噴霧除臭稀釋100~200倍，每周2~3次
糞便堆肥每噸糞便：EM菌5~10L + 米糠20kg，堆肥15~30天

5.4 堆肥製作

配方：有機物料1噸 + EM菌液5~10L + 紅糖5kg → 調整水分至50~60% → 堆置1~1.5m高 → 覆蓋帆布

時程：7天後翻堆，之後每3~5天翻一次。夏季20~30天、冬季40~60天完全腐熟。

對比傳統：時間縮短50~60%、臭味減少80~90%、養分保留率提升。

5.5 生物防蟲液

配方：EM菌1L + 紅糖1L + 醋精1L + 白酒(52°)1L + 辣椒粉1L + 水10L

混合密封發酵15天（25°C），使用時稀釋200~1000倍噴灑葉面。

捌、真實案例深度拆解

案例一：台南善化紅龍果——黃政德（產銷班班長）

背景：4分地紅龍果園，2016年首次接觸光合菌，原慣行農法全面轉用

操作：20L透明桶擴培（食工所母菌+培養基+RO水），每周稀釋300~500倍灌根 + 葉面噴施每10~14天一次

關鍵試驗：隔壁行不施肥（僅光合菌）vs 對照組慣行施肥

指標使用光合菌對照組
單果重2公斤0.8~1.2公斤
產期延長至冬季正常季節
年肥料費12,000元60,000元
肥料用量1/51

教訓：根莖類作物膨大期禁止使用；最佳時機為開花前促進根系與花芽分化。

案例二：屏東芒果——林毓健

背景：2018年食工所課程後導入。此前市售光合菌20L = 2,500元；自製成本 < 200元。

操作：稀釋300~500倍月2次灌根，避開開花期，不混殺菌劑。

成效：芒果樹勢健康、葉片旺盛。連續3年。認為「市售光合菌很混亂，不敢亂買」。

案例三：中國EM菌養豬場（來源：壹讀）

指標改善幅度
料肉比下降15%
氨氣濃度下降60~80%
死亡率降低30~50%
獸藥費用節省40~60%

水產應用：每畝每周潑灑1kg菌液；飼料添加100~200ml EM菌+糖200g+水15kg+飼料100kg。

案例四：屏東縣政府推廣光合菌

屏東縣政府農業處邀請食工所廖麗玲博士指導，推動友善農業。光合菌具固氮溶磷作用，增加土壤益菌量，提高作物吸收力與品質，達到農藥及肥料減量。

案例五：雲林農友寶哥

比喻：「光合菌就像益生菌，腸道健康才能良好吸收。」參加食工所課程後在雲林推廣。

玖、常見失敗原因與排除

培養失敗

現象原因解決方案
菌液發黑/發臭雜菌污染重新培養，加強消毒，使用RO水
菌液不變紅光照不足 / 溫度太低移至陽光充足處，保持25°C以上
菌液變綠藻類污染加強密封避光，檢討水源
產氣過多/爆桶糖分過高 / 溫度太高降低糖量，控制溫度 < 40°C

田間應用失敗

現象原因解決方案
無明顯效果菌液濃度不足 / 頻率太低提高至200~300倍，縮短至7天一次
病害增加混入病原菌送檢菌液，重新培養，加強無菌操作
葉片灼傷稀釋倍數 < 100倍 / 中午噴施調整至300倍以上，傍晚噴施

拾壹、風險與注意事項

  1. 光合菌不是肥料：它分解養分而非創造養分，需搭配基礎施肥
  2. 菌液保存期限：EM菌1個月內最佳（2個月後活菌降50%）；光合菌3~6個月
  3. 殺菌劑禁忌：不可與殺菌劑、抗生素混合使用
  4. 法規現況：光合菌尚未正式登記為微生物肥料（非孢子菌保存不易），市售品質參差
  5. 根莖類限制：地下部膨大期禁用光合菌
  6. 開花期限制：光合菌盛花期暫停
  7. 雜菌污染：自製菌液可能混入病原菌，需加強消毒
  8. 購買建議：向食工所等可信來源取得母菌，避免來路不明產品

拾貳、總結效益矩陣

  • 土壤改良：全面性
  • 水產：良好
  • 畜禽除臭：極佳

  • 成本效益：極高

  • 水產：極佳（降氨氮）
  • 畜禽除臭：良好
  • 堆肥加速：普通
  • 保存：較佳（厭氧性穩定）

最終建議

以3~5公頃農場為例：

  • 年淨效益：8~15萬元（肥料減少+產量提升-材料成本）
  • 首次投入：低於2,000元
  • 月增人力：40~60工時（分散於現有勞動力）
  • 最高策略：光合菌為主（根系強化）+ EM菌為輔（堆肥+除臭）
  • 入門行動：先向食品工業發展研究所參加免費課程，取得正確母菌與技術
